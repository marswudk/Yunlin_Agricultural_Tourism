### 發想
*每到放假就要出遊去，去花東看山看海、到台南大啖美食、去墾丁戲水、去南投享受山林之美，全台灣跑透透的你，為何總獨漏了一個地方呢？那就是雲林！
這個小地方看似平凡無奇，或許連在地人都不一定好好探索過，但事實上，這裡可是藏著許多其他縣市都看不到的美景啊！讓我們來認識這個純樸的好地方，美卻不擠，許多在地人看完一定大呼：「原來雲林還有這麽漂亮的地方！」*
　　在雜誌上看到這麼一篇文章，才想到原來自己對家鄉如此不熟悉。身為一個雲林農家子弟，每每放假回家要不去田裡幫農，要不在家窩著，再不然和朋友出去也總是在街上晃悠著，沒想過要到哪裡找點新鮮。
　　雲林為台灣農業首都，每當朋友一知道我家裡務農，總會對我家裡的溫室作物感到興趣，老是嚷嚷著要來採番茄，於是想到可以透過農業體驗，像大湖採草莓一樣，來介紹雲林縣的觀光路線，也許可以讓更多人認識農業、對農業、對雲林產生興趣。
　　透過這個專題實作，結合課堂所學之前端、後端的技術，來嘗試做出一個雲林觀光網站，介紹一些實際現有的農業體驗活動，並虛擬做出一個雲林觀光護照─Yunlin Pass，專題就此產生。
